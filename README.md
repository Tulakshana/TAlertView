## Usage

```objc
[TWAlertView showAlert:@"Hello" message:@"Your message goes here..." callback:nil];
```

## Installation

TWAlertView is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "TWAlertView"
```
